# 🗄️ SQL Query Generator

## 📌 Project Description
SQL Query Generator is a Python-based tool that automatically generates SQL queries based on user input.  
The application provides a simple interface where users can enter table details and conditions to generate SQL queries such as **SELECT, INSERT, UPDATE, and DELETE**.

This tool helps beginners and developers quickly create SQL queries without writing them manually.

---

## 🎯 Features
- Generate **SELECT queries**
- Generate **INSERT queries**
- Generate **UPDATE queries**
- Generate **DELETE queries**
- Simple and user-friendly interface
- Reduces manual SQL coding errors
- Fast query generation

---

## ⚙️ Technologies Used
- Python
- Gradio
- SQLParse
- Pandas
